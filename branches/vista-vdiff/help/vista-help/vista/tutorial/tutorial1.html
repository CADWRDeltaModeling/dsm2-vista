Vista Tutorial

Getting Started

  1. Click on Window's Start Button. Look for Vista Folder and click on Vista. If you are looking at this help you are probably already familiar with this step.

  1. This opens a window on the screen with title "DATA VISTA (Vista ??: Build date dd-MM-yy). This should have four menu items, Session, Group, Options and Help.

  2. To get started, click on the Session Menu Item.




  1. Click on the DSS File item under Open Menu Item.


    1. This will open a dialog box for selecting files. Select a dss file by browsing or enter the full relative pathname of the file and click Open.


    2. This will open a "Group" Frame which displays all the data sets available in this dss file



    3. Click on the pathnames in the table as shown above and then select from the Data Menu , the Show As... item from which select Graph item from which select the Time Series item

    4. This will display the selected item or items in a Graph.


    5. Try out the Data|ShowAs|Table ( The | symbol separates hierarchy )

    6. DSS data can be exported to text files or to other DSS Files using the Data|Export Data To...| sub items

    7. The Data|Delete menu item clears from this table view the selected items

    8. The Data|Reload Group item reloads the current group by recataloging the DSS file.
