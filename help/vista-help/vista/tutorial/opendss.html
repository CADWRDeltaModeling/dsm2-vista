Opening a dss file

A dss file is a data base of time series. These time series are represented by their names an characteristics. These characteristics are embodies in a pathname.

A pathname is a unique name given to a time series for storing it in a DSS database file. The pathname consists of 6 parts named from A-F. Parts A,B,C and F are user defined while part D is reserved for the time window and the E part for the time interval.
